ReadMe for Auction Server

1. Compile the ServerStart.java file
2. Run ServerStart.java

then you will get the gui including the list of items in stock exchange with thier current prices.As in the guideline the list only includes the following symbols; 
FB, VRTU, MSFT, GOOGL, YHOO, XLNX, TSLA and TXN.

Then you can use nc to connect the server using the command 'nc ipadress 2000'. You are asked to enter the name, then the symbol and then the your bid.
Here date is the current date and time is current time. Here auction txts contains the changes that are done by the bidders to the stock items. And also you can filter the symbols and get the history of each selected symbols. All the details are displayed in the gui.
